WordPlay

Overview:
Wordplay is a cutting-edge programming platform that enables accessible, interactive typography while teaching youth from diverse linguistic backgrounds how to code. As a contributor, I focused on enhancing user accessibility, interactive design, and platform performance.


Key Contributions:
- Interactive Design & Accessibility:
Leveraged React and Figma to develop accessible, interactive features for typography, supporting educational goals for multilingual youth.
- Platform Development & Maintenance:
Utilized HTML, CSS, JavaScript, TypeScript, and Svelte/SvelteKit to build, enhance, and maintain the open-source platform. Implemented real-time functionality, addressed defects, and optimized user experiences through scalable, clean code.
- Collaborative Development:
Employed the GitHub Flow strategy for issue tracking, feature branching, and iterative feedback. Developed branches and draft pull requests for new features and bug resolutions, collaborating effectively with maintainers and contributors.

Additional Information

Technologies Used:
Languages & Frameworks: React, TypeScript, JavaScript, HTML, CSS, Svelte/SvelteKit
Design Tools: Figma
Version Control: GitHub

Impact: This role highlighted my ability to collaborate within a dynamic, open-source environment while delivering accessible and innovative solutions to empower young learners through creative coding. Through thoughtful feature development and rigorous iteration, I helped transform complex technical requirements into user-friendly solutions that encouraged inclusivity and creativity. Working in this open-source setting deepened my understanding of community-driven development, the importance of transparency, and the value of iterative feedback.
